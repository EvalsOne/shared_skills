# Shared Skills

A collection of reusable Agent Skills for [ConsoleX AI](https://consolex.ai).

## Structure

```
shared_skills/
├── <skill-name>/          # Skill source directory
│   ├── SKILL.md           # Skill definition and instructions
│   └── references/        # Supporting files (templates, schemas, etc.)
└── packages/              # Packaged skill files (.skill)
```

## Available Skills

| Skill | Description |
|-------|-------------|
| `doc-to-slides-images` | Convert documents to slide image specifications |

## Usage

Skills in this repository can be imported into ConsoleX AI agents to extend their capabilities.
