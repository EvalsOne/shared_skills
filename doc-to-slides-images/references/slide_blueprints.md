# Slide Blueprints (choose one per slide)

- TITLE: Big title + short subtitle + small icon
- AGENDA: 4–6 agenda items, numbered, minimal icons
- SECTION: Section header slide, one strong visual metaphor
- CONTENT_LEFT_VISUAL_RIGHT: Left bullets, right illustration/diagram
- VISUAL_LEFT_CONTENT_RIGHT: Left illustration, right bullets
- TWO_COLUMN: Two columns of bullets (compare or categorize)
- TIMELINE: Horizontal timeline with 4–6 milestones
- PROCESS: 4–6 step flow with arrows
- COMPARISON: Table-like comparison (2–3 columns max)
- METRICS: 3–6 key metrics cards with tiny icons
- CHECKLIST: Checklist with tick icons
- QUOTE: One statement + attribution (if known)
- SUMMARY: 5 key takeaways
- CTA: Next steps / actions (only if doc type is proposal/plan)
