# Style Router

Default: NotebookLM clean deck:
- light background, high contrast
- flat icons, subtle dividers
- 2–3 color palette
- big title, short bullets

Doc-type routing (override default when confident):
- Academic/technical paper: cleaner, more diagram/flowchart, fewer icons, more structure (process/timeline/metrics)
- Product/marketing page: bolder accent color, more “cards”, CTA slide allowed
- Financial/report: metrics cards, bar/line charts, restrained palette
- Tutorial/how-to: checklist + process slides, step numbers prominent
- Narrative essay/op-ed: quote + section dividers, editorial illustration

User style instruction always wins.
